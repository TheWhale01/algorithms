# Algorithms - Linked Lists C
This folder contains a definition of small function utilities to work with linked
lists in C

## How to compile ?
1. *NOT RECOMENDED*
```bash
gcc *.c
./a.out
```

2.
```bash
mkdir build/
cd build/
cmake ../
make
./linked_lists
```

Default compile options with CMake are `-Wall -Wextra -Werror -g`
