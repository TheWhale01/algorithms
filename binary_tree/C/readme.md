# Algorithms - Linked Lists C
This folder contains a definition of small function utilities to work with binary
trees in C

## How to compile ?
1. *NOT RECOMENDED*
```bash
gcc *.c
./a.out
```

2.
```bash
mkdir build/
cd build/
cmake ../
make
./binary_tree
```

Default compile options with CMake are `-Wall -Wextra -Werror -g`
